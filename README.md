# Fast-Modular-Exponentiation
Different implementations to calculate the Fast Modular Exponentiation using:
- Right-to-Left Binary Modular Exponentiation.
- Left-to-Right Binary Modular Exponentiation.
- CRT.
- CRT Parallelized.
